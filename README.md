## 💡 Motivation

Windows shortcut files (`.lnk`) are everywhere, but Windows makes it surprisingly difficult to inspect or modify them properly. If you right-click a shortcut and open its properties, the OS only lets you change a few basic fields. And when a shortcut breaks, points to a missing network drive, or needs a batch update across multiple deployments, fixing it manually via the Windows UI is tedious and limited. Developers and power users often have to resort to writing custom scripts or using outdated tools just to see what is actually hidden inside a shell link file.

**Easy Link File Viewer** was built to provide a clean, dedicated workspace to handle shortcut files. It exposes properties of a `.lnk` file in a friendly user interface—including a built-in hexadecimal viewer for raw structure analysis and an advanced icon changer—making it simple to inspect, repair, and rewrite shortcuts with complete precision.

##### ⚡ The Real Question
###### Why struggle with the restricted Windows properties menu to guess what is inside a shortcut, when you can easily view, modify, and fix every single low-level property of a `.lnk` file in a dedicated, portable tool?.

## 🤖 Features

- Portable application. Just copy the software anywhere.
- Supports multiple program instances. You can open and edit various files at once this way.
- Friendly file editor.
- Hexadecimal Viewer (read-only).
- Shortcut's icon changer.
- File Drag&Drop.
- Visual Theming.
- List with Most Recent Used (MRU) shortcut files.
- Can read shortcuts that points to special targets.
- Apply changes directly to the current shortcut file, or save a copy of it to another destination.

## 📝 Requirements

- Microsoft Windows OS.

## 🚀 Getting Started

1. Navigate to the **[Releases page](https://github.com/ElektroStudios/Easy-Link-File-Viewer/releases/latest)**.
2. Download the latest `.zip` archive or the `.exe` setup installer, depending on your preference.
3. If you downloaded the `.zip` archive, extract its contents to your preferred directory.
   <br> 
   If you downloaded the `.exe` file, run it and follow the installation wizard.
4. Run the executable file to launch the application.
